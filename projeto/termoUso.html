Termo de Uso e Política de Privacidade

Última atualização: Dezembro/2025

1. Resumo

Este Termo de Uso e Política de Privacidade tem como finalidade esclarecer como tratamos seus dados pessoais ao utilizar a plataforma Amaná, que permite criar uma conta, registrar um dispositivo (ESP32) e enviar dados para armazenamento e consulta.

Ao criar uma conta, você concorda com estes termos e autoriza o tratamento dos seus dados conforme descrito abaixo.

2. Dados Coletados

2.1. Dados fornecidos pelo usuário

  • Nome
  • E-mail
  • Senha (armazenada de forma criptografada)

2.2. Dados do dispositivo

  • Endereço MAC
  • Dados enviados pelo dispositivo (ex.: sensores)
  • Endereço IP utilizado para envio

2.3. Dados gerados automaticamente

  • Logs de acesso
  • Data e hora das requisições
  • Informações de uso do sistema

3. Finalidades do Tratamento

Seus dados são utilizados para:

  • Criar e gerenciar sua conta
  • Garantir autenticação e segurança
  • Associar dispositivos
  • Armazenar e disponibilizar dados
  • Gerar gráficos e análises
  • Manter o funcionamento e segurança do sistema
  • Enviar comunicações essenciais

Não utilizamos seus dados para finalidades não previstas neste termo.

4. Compartilhamento de Dados

Seus dados não são vendidos. Dados marcados por você como públicos poderão ser exibidos para outros usuários.

Dados privados só serão compartilhados em caso de:

  • Obrigação legal
  • Ordem judicial

5. Base Legal (LGPD)

  • Art. 7, I — Consentimento do titular
  • Art. 7, IX — Legítimo interesse

6. Segurança dos Dados

  • Criptografia de senhas
  • Controle de acesso
  • Logs de auditoria
  • Monitoramento de tentativas de acesso indevido

7. Direitos do Usuário

  • Acesso aos seus dados
  • Correção
  • Exclusão
  • Portabilidade
  • Revogação do consentimento

8. Retenção dos Dados

  • Dados pessoais eliminados em até 90 dias
  • Dados de dispositivos podem ser anonimizados ou excluídos

9. Responsabilidades do Usuário

  • Fornecer informações verdadeiras
  • Não compartilhar senha
  • Uso lícito do sistema
  • Não explorar vulnerabilidades
  • Enviar certificado de calibração (quando aplicável)

10. Alterações deste Termo

Você será notificado em caso de mudanças relevantes.

11. Aceite

  • Li e aceito os Termos de Uso e Política de Privacidade
  • Autorizo o tratamento dos dados conforme descrito
